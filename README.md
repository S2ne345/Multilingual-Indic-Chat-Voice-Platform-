# Multilingual-Indic-Chat-Voice-Platform-
A conversational AI system that supports Tamil, Hindi, and English (including code-mixed queries) via text and voice. Built with FastAPI, it uses Retrieval-Augmented Generation (RAG) for grounded answers with citations, and includes pluggable adapters for ASR/TTS/LLM (mock by default, ready to integrate providers such as Azure Speech and OpenAI).
